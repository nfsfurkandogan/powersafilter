# Powersa Filter Web

Bu klasör, Powersa Filter için statik bir prototip içerir.

## Çalıştırma

Yerel bir sunucu üzerinden açmanız önerilir.

```bash
python3 -m http.server 8080
```

Tarayıcıda `http://localhost:8080` adresini açın.

## Veri Güncelleme

Ürün listesi `data/products.js` dosyasındadır. Excel güncellenirse dosyayı yeniden üretmek gerekir.
